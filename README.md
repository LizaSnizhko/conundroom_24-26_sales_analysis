Project Overview

This project analyzes sales and booking data for Conundroom Escape Rooms from July 1, 2024 through June 30, 2026. The goal is to explore booking trends, revenue, customer behavior, and business performance across all three locations using SQL and Tableau.

The analysis uses data exported from the Bookeo booking platform for all three Conundroom locations. The dataset contains booking information, customer information, and revenue-related fields for a two-year period.

Some additional data preparation was required before importing the files into PostgreSQL. Data from multiple quarterly exports was combined, location-specific columns were standardized, and all bookings were merged into a single dataset.

The analysis uses data exported from the Bookeo booking platform and Square for three Conundroom locations. The datasets contain booking information, customer information, and payments for 2 years.
